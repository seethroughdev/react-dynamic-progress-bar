React State Progress Bar
Demo Download Usage Github

Add a loading timer to your ajax calls, page transitions or anything you want.

Start and finish a loading bar without knowing the actual value. It just works.

Most progress bars require you to input a value so the loading bar can calculate the width. But with ajax calls and other things, we don't always know how long it's going to take.

React State Progress Bar only requires a boolean state, either you start the bar, and its active. Or you cancel the bar and it finishes up. bar = true, or bar = false.

Download

React State Progress Bar is browserify compatible, and uses UMD. So its available for commonJS, AMD or in the browser. SystemJS version will be coming soon.

Usage

Use React State Progress Bar like any other react component. Load the file, and include the component!

With NPM:


// require the file
var ProgressBar = require('react-state-progress-bar');

// inside your React component
render: function() {
    return (
        <ProgressBar bar={this.state.bar} />
    )
}

If the bar prop is not filled out, the component will consider it null and wait to start the bar until you change it to true. Afterwards, change the props to false, and the bar will finish up.

Options

There are several props you can pass to the progress bar, only bar is required. 


<ProgressBar
    bar={this.state.bar}
    barColor='#990000'
    barHeight='10'
    barId='myNewBarId'
    barClass='myNewBarClass'
    barCallback={this.onBarComplete}
/>
  • bar (boolean) * required | pass a true or false value to bar to turn the bar on and off.
  • barColor (string) | pass a color code for the bar. #990000 or rgba(0,0,0,1)
  • barHeight (string) | pass a number string in pixels for the height of the bar.
    You only need to pass the number, right now, only px are supported.
  • barPosition (string) | pass the word 'bottom' here to position the bar across the bottom of the screen.
    Also note, you can override the entire position with the barStyle prop.
  • barStyle (object) | pass a style object in and it will be merged with the existing styles.
    Also note, this is a shallow merge, and will replace all existing keys with your own.
